# Disable Regen

[Power Type](../power_types.md).

Disables the natural health regeneration when on full hunger.

Type ID: `origins:disable_regen`

### Fields

_None._


### Example
```json
{
    "type": "origins:disable_regen"
}
```
This power disables natural regeneration of the player.
