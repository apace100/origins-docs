# Ignore Water

[Power Type](../power_types.md).

Prevents the movement of the player from being affected by water.

Type ID: `origins:ignore_water`

### Fields

_None._

### Example
```json
{
  	"type": "origins:ignore_water"
}
```
This power makes the player ignore water. :)
