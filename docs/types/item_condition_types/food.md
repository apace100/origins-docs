# Food

[Item Condition Type](../item_condition_types.md)

Checks whether the item is a food component.

Type ID: `origins:food`


### Fields

_None._


### Examples

```json
"item_condition": {
    "type": "origins:food"
}
```

This example will check if the item is considered a food item.
