# Food

[Item Condition Type](../item_condition_types.md)

Checks whether the item is a food component.

Type ID: `origins:food`

### Fields:

_None._

### Example
```json
"item_condition": {
    "type": "origins:food"
}
```
This example checks if the item is a food item.
