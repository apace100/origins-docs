# Fall Flying

[Entity Condition Type](../entity_condition_types.md)

Checks whether the player is currently fall flying (with an Elytra, an Elytra power, or similar).

Type ID: `origins:fall_flying`

### Fields:

_None._

### Example:
```json
"condition": {
    "type": "origins:fall_flying"
}
```
