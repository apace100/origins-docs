# Sprinting

[Entity Condition Type](../entity_condition_types.md)

Checks whether the entity is currently sprinting.

Type ID: `apoli:sprinting`

!!! note

    **This entity condition type will only work on players.**


### Fields

_None._


### Examples

```json
"condition": {
    "type": "apoli:sprinting"
}
```
