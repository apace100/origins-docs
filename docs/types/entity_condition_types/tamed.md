# Tamed

[Entity Condition Type](../entity_condition_types.md)

Checks whether a tamable entity is tamed. Will always return false if the entity is not tamable.

Type ID: `apoli:tamed`

### Fields

_None._

### Examples

```json
"condition": {
    "type": "apoli:tamed"
}
```
