# Invisible

[Entity Condition Type](../entity_condition_types.md)

Checks whether the player is currently invisible - should capture all sources of invisibility.

Type ID: `origins:invisible`

### Fields:

_None._

### Example:
```json
"condition": {
    "type": "origins:invisible"
}
```
