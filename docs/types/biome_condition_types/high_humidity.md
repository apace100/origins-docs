# High Humidity

[Biome Condition Type](../biome_condition_types.md)

Checks whether the biome counts as an area with high humidity.

Type ID: `origins:high_humidity`

### Fields:

_None._

### Example:
```json
"biome_condition": {
    "type": "origins:high_humidity"
}
```
