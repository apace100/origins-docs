# Owner

[Bi-entity Condition Type](../bientity_condition_types.md)

Checks whether the actor entity is the owner of the tamable target entity.

Type ID: `origins:owner`


### Fields

_None._


### Examples

"bientity_condition": {
    "type": "origins:owner"
}
