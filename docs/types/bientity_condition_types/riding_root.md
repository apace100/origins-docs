# Riding Root

[Bi-entity Condition Type](../bientity_condition_types.md)

Checks if the actor entity is riding the target entity from the very end of the riding chain.

Type ID: `origins:riding_root`


### Fields

_None._


### Examples

```json
"bientity_condition": {
	"type": "origins:riding_root"
}
```
