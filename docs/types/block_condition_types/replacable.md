# Replacable

[Block Condition Type](../block_condition_types.md)

Checks whether the block is a replacable material (e.g. tall grass, water, ...).

Type ID: `origins:replacable`


### Fields

_None._


### Examples

```json
"block_condition": {
    "type": "origins:replacable"
}
```
