# Tame

[Bi-entity Action Type](../bientity_action_types.md)

Tames the target entity with the actor entity as the owner. Only works on tameable entities such as wolves or cats.

Type ID: `apoli:tame`

### Fields

_None._

### Examples

```json
"bientity_action": {
    "type": "apoli:tame"
}
```
