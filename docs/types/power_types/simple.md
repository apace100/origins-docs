# Simple

[Power Type](../power_types.md)

Does nothing.

Type ID: `apoli:simple`

!!! note

    This power type can be used by <u>**addon**</u> developers by creating a new `PowerTypeReference` to provide functionality.

### Fields

_None._

### Examples

```json
{
	"type": "apoli:simple"
}
```

This example will do nothing.
