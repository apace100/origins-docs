# Still

[Fluid Condition Type](../fluid_condition_types.md)

Checks whether this state describes a still (i.e. not flowing, i.e. source) fluid.

Type ID: `origins:still`

### Fields:

_None._

### Example:
```json
"fluid_condition": {
    "type": "origins:still"
}
```
