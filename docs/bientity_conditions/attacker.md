# Attacker

[Bi-entity Condition](../bientity_conditions.md)

Checks if the actor entity has attacked the target entity within the last 5 seconds, and still exists.

Type ID: `origins:attacker`

### Fields

_None._

### Example

```json
"bientity_condition": {
  "type": "origins:attacker"
}
```
